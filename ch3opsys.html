Operating System Terminology
Term Definition
Operating System Job Queue The collection of all processes in the system waiting for processing.
FIFO First In, First Out; a scheduling method where the first process in the queue is the first to be processed.
Interactive Processing A method of processing data in which the user can interact with the computer while the processing takes place.
Time-Sharing A multitasking approach where multiple users can access a system seemingly simultaneously by dividing processor time.
Multiprogramming Running multiple programs on a computer system at the same time by sharing system resources.
Application Software Software designed to perform specific tasks for users, such as word processing or web browsing.
System Software Software that manages and controls the hardware components of a computer system.
Utility Software System management tools, like antivirus programs and disk management software, that support and enhance the operating system's functionality.
Graphical User Interface (GUI) A user interface that allows users to interact with electronic devices through graphical icons and visual indicators.
Shell A command-line interface or graphical interface that provides access to the services of the operating system.
Kernel The core part of an operating system, responsible for managing system resources and communication between hardware and software.
Device Driver Software that allows the operating system to communicate with hardware devices.
File Manager A system software component that manages files and directories on a computer.
Memory Manager Part of the operating system that manages system memory, allocating and deallocating memory space as needed.
Virtual Memory A memory management technique that uses a portion of the hard drive as an extension of RAM.
Bootstrapping The process of starting a computer and loading the operating system into memory.
ROM Read-Only Memory; a type of non-volatile memory used to store firmware or software that is rarely changed.
Deadlock A situation where two or more processes cannot proceed because each is waiting for the other to release resources.
Spooling A process of managing data by placing it in a temporary storage area until it can be processed.
Super User (Administrator) A user with special privileges that allow full control over a computer system.
Multi-Factor Authentication An authentication method that requires the user to provide two or more verification factors to gain access to a resource.
