Jarullah Bana

Term Definition
LAN Local Area Network: A network that connects computers within a limited area like a residence or office building.
MAN Metropolitan Area Network: A network that connects users within a region like a city.
WAN Wide Area Network: A telecommunications network that extends over a large geographical area.
CSMA/CD Ethernet Carrier Sense Multiple Access with Collision Detection: A protocol used in Ethernet technology to manage packet collisions on a network.
CSMA/CA WiFi Carrier Sense Multiple Access with Collision Avoidance: A protocol used in wireless networks to avoid collisions.
Repeater A device that receives and amplifies a signal to extend the reach of a network.
Router A device that forwards data packets between computer networks, creating an overlay network.
ICANN ICANN: The Internet Corporation for Assigned Names and Numbers, responsible for coordinating domain names and IP addresses globally.
Country-code TLDs Country-code TLDs: Two-letter top-level domains that represent specific countries or regions, such as .us for the United States or .uk for the United Kingdom.
Client/Server Model A network structure where clients request resources from centralized servers.
P2P: Peer-to-Peer A decentralized network model where peers share resources directly with each other without a central server.
IP Address An Internet Protocol address is a numerical label assigned to devices connected to a computer network.
Firewall A network security device that monitors and filters incoming and outgoing network traffic based on security rules.
Antivirus Software Software designed to detect and remove viruses and other malicious software from computers.
